# XRay Mod for Minecraft 1.21.11

Простой и эффективный X-Ray мод для Minecraft 1.21.11 с поддержкой обхода anti-xray плагинов.

## 📋 Требования

- **Java 17+** - [Скачать](https://adoptium.net/)
- **Fabric Loader** для 1.21.11
- **Minecraft 1.21.11**

## 🚀 Быстрый старт

### 1. Скачай готовый мод (JAR файл)

Перейди в раздел [Releases](https://github.com/interiumcrack/xraymod-1.21.11/releases) и скачай `xraymod-1.0.0.jar`

### 2. Установи мод

1. Открой Minecraft Launcher
2. Перейди в папку `.minecraft`:
   - **Windows**: `%AppData%\.minecraft`
   - **Mac**: `~/Library/Application Support/minecraft`
   - **Linux**: `~/.minecraft`
3. Перейди в папку `mods/` (создай если нет)
4. Скопируй туда `xraymod-1.0.0.jar`
5. Перезагрузи Minecraft

### 3. Используй в игре

- Нажми **X** чтобы включить/отключить X-Ray
- В чате появится сообщение:
  - `X-Ray: Включен` (зелёный)
  - `X-Ray: Выключен` (красный)

## 🔧 Компиляция из исходников

Если хочешь самостоятельно скомпилировать мод:

```bash
# Клонируй репозиторий
git clone https://github.com/interiumcrack/xraymod-1.21.11.git
cd xraymod-1.21.11

# Скомпилируй на Windows
gradlew.bat build

# Или на Linux/Mac
./gradlew build
```

JAR файл будет в папке `build/libs/xraymod-1.0.0.jar`
